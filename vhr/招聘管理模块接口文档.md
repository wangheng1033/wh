# 招聘管理模块接口文档

## 一、简历管理接口

### 1. 分页查询简历列表
- **URL**: `/recruit/resume/`
- **Method**: GET
- **参数**:
  - page: 页码，默认1
  - size: 每页数量，默认10
  - keyword: 搜索关键词（姓名或电话）
  - education: 学历筛选
  - status: 状态筛选（0-待筛选，1-已筛选，2-待面试，3-已拒绝，4-已录用）
  - workYears: 工作年限筛选
- **响应**: RespPageBean

### 2. 添加简历
- **URL**: `/recruit/resume/`
- **Method**: POST
- **参数**: Resume对象
  - name: 姓名（必填）
  - gender: 性别（必填）
  - age: 年龄
  - education: 学历
  - workYears: 工作年限
  - phone: 联系电话（必填）
  - email: 邮箱
  - expectedSalary: 期望薪资
  - jobIntention: 求职意向（必填）
  - workExperience: 工作经历
  - educationExperience: 教育经历
  - attachmentUrl: 附件URL
  - attachmentName: 附件名称
  - remark: 备注
- **响应**: RespBean

### 3. 更新简历
- **URL**: `/recruit/resume/`
- **Method**: PUT
- **参数**: Resume对象（需包含id）
- **响应**: RespBean

### 4. 删除简历
- **URL**: `/recruit/resume/{id}`
- **Method**: DELETE
- **参数**: id-简历ID
- **响应**: RespBean

### 5. 根据ID查询简历
- **URL**: `/recruit/resume/{id}`
- **Method**: GET
- **参数**: id-简历ID
- **响应**: Resume对象

### 6. 更新简历状态
- **URL**: `/recruit/resume/status/{id}`
- **Method**: PUT
- **参数**:
  - id: 简历ID
  - status: 状态值（0-待筛选，1-已筛选，2-待面试，3-已拒绝，4-已录用）
- **响应**: RespBean

### 7. 根据状态查询简历
- **URL**: `/recruit/resume/status/{status}`
- **Method**: GET
- **参数**: status-状态值
- **响应**: Resume列表

### 8. 导出简历数据
- **URL**: `/recruit/resume/export`
- **Method**: GET
- **响应**: Excel文件

### 9. 批量导入简历
- **URL**: `/recruit/resume/import`
- **Method**: POST
- **参数**: file-Excel文件
- **响应**: RespBean

---

## 二、面试管理接口

### 1. 分页查询面试列表
- **URL**: `/recruit/interview/`
- **Method**: GET
- **参数**:
  - page: 页码，默认1
  - size: 每页数量，默认10
  - candidateName: 候选人姓名
  - positionId: 岗位ID
  - interviewerId: 面试官ID
  - interviewType: 面试类型（1-初面，2-复面，3-终面）
  - status: 状态（0-待面试，1-已通过，2-未通过，3-待定）
- **响应**: RespPageBean

### 2. 添加面试安排
- **URL**: `/recruit/interview/`
- **Method**: POST
- **参数**: Interview对象
  - resumeId: 简历ID（必填）
  - candidateName: 候选人姓名
  - positionId: 岗位ID（必填）
  - positionName: 岗位名称
  - interviewTime: 面试时间（必填）
  - interviewLocation: 面试地点（必填）
  - interviewerId: 面试官ID（必填）
  - interviewerName: 面试官姓名
  - interviewType: 面试类型（1-初面，2-复面，3-终面）（必填）
  - remark: 备注
- **响应**: RespBean

### 3. 批量添加面试
- **URL**: `/recruit/interview/batch`
- **Method**: POST
- **参数**: Interview对象列表
- **响应**: RespBean

### 4. 更新面试
- **URL**: `/recruit/interview/`
- **Method**: PUT
- **参数**: Interview对象（需包含id）
- **响应**: RespBean

### 5. 删除面试
- **URL**: `/recruit/interview/{id}`
- **Method**: DELETE
- **参数**: id-面试ID
- **响应**: RespBean

### 6. 根据ID查询面试
- **URL**: `/recruit/interview/{id}`
- **Method**: GET
- **参数**: id-面试ID
- **响应**: Interview对象

### 7. 根据简历ID查询面试
- **URL**: `/recruit/interview/resume/{resumeId}`
- **Method**: GET
- **参数**: resumeId-简历ID
- **响应**: Interview列表

### 8. 更新面试状态
- **URL**: `/recruit/interview/status/{id}`
- **Method**: PUT
- **参数**:
  - id: 面试ID
  - status: 状态值（0-待面试，1-已通过，2-未通过，3-待定）
- **响应**: RespBean

### 9. 填写面试评价
- **URL**: `/recruit/interview/evaluation/{id}`
- **Method**: PUT
- **参数**:
  - id: 面试ID
  - score: 评分（0-100）
  - advantages: 优点评价
  - disadvantages: 缺点评价
  - suggestion: 录用建议（0-待定，1-建议录用，2-不建议录用）
  - evaluation: 综合评价
  - status: 面试结果状态
- **响应**: RespBean

### 10. 面试统计
- **URL**: `/recruit/interview/statistics`
- **Method**: GET
- **参数**:
  - startDate: 开始日期（yyyy-MM-dd）
  - endDate: 结束日期（yyyy-MM-dd）
- **响应**: 统计列表（岗位名称、面试人数、通过人数、未通过人数）

## 三、Offer管理接口

### 1. 分页查询Offer列表
- **URL**: `/recruit/offer/`
- **Method**: GET
- **参数**:
  - page: 页码，默认1
  - size: 每页数量，默认10
  - candidateName: 候选人姓名
  - positionId: 岗位ID
  - status: 状态（0-待确认，1-已发送，2-已接受，3-已拒绝，4-已过期）
  - viewStatus: 查看状态（0-未查看，1-已查看）
- **响应**: RespPageBean

### 2. 添加Offer
- **URL**: `/recruit/offer/`
- **Method**: POST
- **参数**: Offer对象
  - resumeId: 简历ID（必填）
  - interviewId: 面试ID
  - candidateName: 候选人姓名
  - positionId: 岗位ID
  - positionName: 岗位名称
  - salary: 薪资（必填）
  - probationPeriod: 试用期（月），默认3（必填）
  - entryDate: 入职日期（必填）
  - offerValidDate: Offer有效期（必填）
  - remark: 备注
- **响应**: RespBean

### 3. 更新Offer
- **URL**: `/recruit/offer/`
- **Method**: PUT
- **参数**: Offer对象（需包含id）
- **响应**: RespBean

### 4. 删除Offer
- **URL**: `/recruit/offer/{id}`
- **Method**: DELETE
- **参数**: id-OfferID
- **响应**: RespBean

### 5. 根据ID查询Offer
- **URL**: `/recruit/offer/{id}`
- **Method**: GET
- **参数**: id-OfferID
- **响应**: Offer对象

### 6. 根据简历ID查询Offer
- **URL**: `/recruit/offer/resume/{resumeId}`
- **Method**: GET
- **参数**: resumeId-简历ID
- **响应**: Offer对象

### 7. 发送Offer
- **URL**: `/recruit/offer/send/{id}`
- **Method**: PUT
- **参数**: id-OfferID
- **响应**: RespBean

### 8. 更新Offer状态
- **URL**: `/recruit/offer/status/{id}`
- **Method**: PUT
- **参数**:
  - id: OfferID
  - status: 状态值（0-待确认，1-已发送，2-已接受，3-已拒绝，4-已过期）
  - rejectReason: 拒绝原因（状态为3时填写）
- **响应**: RespBean

### 9. 更新查看状态
- **URL**: `/recruit/offer/view/{id}`
- **Method**: PUT
- **参数**: id-OfferID
- **响应**: RespBean

### 10. 查询即将过期的Offer
- **URL**: `/recruit/offer/expiring`
- **Method**: GET
- **参数**: date-日期（yyyy-MM-dd）
- **响应**: Offer列表

### 11. 查询即将入职的Offer
- **URL**: `/recruit/offer/upcoming`
- **Method**: GET
- **参数**:
  - startDate: 开始日期（yyyy-MM-dd）
  - endDate: 结束日期（yyyy-MM-dd）
- **响应**: Offer列表
